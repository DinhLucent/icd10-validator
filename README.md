# icd10-validator

![Python](https://img.shields.io/badge/python-3.9+-blue?logo=python)
![License](https://img.shields.io/badge/License-MIT-green)

Python library for validating ICD-10 codes with format checking, chapter classification, billable code detection, and cross-reference to clinical descriptions

## Features

- Python library for validating ICD-10 codes with format checking, chapter classification, billable code detection, and cross-reference to clinical descriptions

## Installation

```bash
git clone https://github.com/DinhLucent/icd10-validator.git
cd icd10-validator
pip install -r requirements.txt
```
